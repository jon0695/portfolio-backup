 The complete version of my first Python portfolio project HAT Engine
 The "Half-baked Adventure Terminal Engine"

 An easily extensable, very basic, text-adventure engine built in Python

 Dependacies:
 curses (for better terminal support), python 3.11 (or higher)

 Running:
 Execute the 'main.py' file using python to run game.

 Instructions for creating custom pages are in the 'page_tut.py' file
 'page_template.py' can be copied to a new page for a head start.
